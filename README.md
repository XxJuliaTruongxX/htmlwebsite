# Julia Truong's Portfolio Website

A clean, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## About

This is Julia Truong's personal portfolio website showcasing her journey as a Computer Science and Mathematics student at the University of Minnesota. The site features a modern design with smooth scroll animations and a pink/white color scheme.

## Features

- **Responsive Design**: Mobile-first approach ensuring great experience across all devices
- **Smooth Animations**: Scroll-triggered animations using Intersection Observer API
- **Clean Aesthetics**: Minimalist design with custom CSS and Josefin Sans typography
- **Social Integration**: Direct links to GitHub, LinkedIn, and email contact
- **Interactive Elements**: Hover effects and smooth transitions

## Technology Stack

- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Custom styling with flexbox, transitions, and modern features
- **Vanilla JavaScript**: Minimal, performant code for animations
- **Static Hosting**: Deployed on Vercel for fast, reliable hosting

## Project Structure

```
├── index.html          # Main portfolio page
├── styles.css          # All styling and responsive design
├── main.js             # Scroll animations and interactions
├── april_fools.html    # Easter egg page
├── package.json        # Project metadata and scripts
└── assets/             # Images and media files
```

## Development

This is a static website that requires no build process:

```bash
# Install dependencies (for local development server)
npm install

# Start local development server
npm run dev

# Build (no-op for static site)
npm run build
```

## Design Philosophy

- **Simplicity**: Clean, uncluttered interface focusing on content
- **Performance**: Lightweight, fast-loading with minimal dependencies
- **Accessibility**: Semantic HTML and proper contrast ratios
- **Mobile-First**: Responsive design prioritizing mobile experience
